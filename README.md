# Crypto Cassie Generator - GitHub Pages Deployment

This repository contains the static files for the Crypto Cassie Generator web application.

## Deployment Instructions

1. **Create a new GitHub repository** (e.g., `crypto-cassie-generator`)

2. **Upload all files** from this deployment package to your repository:
   - `index.html`
   - `assets/` folder (contains CSS, JS, and image files)

3. **Enable GitHub Pages**:
   - Go to your repository settings
   - Scroll down to "Pages" section
   - Under "Source", select "Deploy from a branch"
   - Choose "main" branch and "/ (root)" folder
   - Click "Save"

4. **Access your site**:
   - Your site will be available at: `https://[your-username].github.io/[repository-name]`
   - It may take a few minutes for the site to become available

## Features

- **Character Consistency**: Maintains Crypto Cassie's visual identity across all generated prompts
- **Scene Presets**: Quick-start templates for popular crypto content scenarios
- **Custom Configuration**: Full control over outfit, scene description, and dialogue
- **Optimized Prompts**: Generates detailed prompts for video generation tools
- **Responsive Design**: Works on desktop and mobile devices

## Usage

1. Select a preset scenario or create your own
2. Customize the outfit style
3. Describe the scene setting
4. Write the dialogue/script
5. Generate the video prompt
6. Copy the prompt and use it with your preferred video generation tool

## Character Profile

**Crypto Cassie** is a hyper-realistic digital influencer designed for crypto content creation:
- **Appearance**: 5'10" Black American woman, green eyes, long wavy dark hair
- **Personality**: Friendly, confident, educational with a playful edge
- **Target Audience**: Crypto enthusiasts, Gen Z, military vets, young investors
- **Platforms**: TikTok, YouTube Shorts

## Technical Details

- Built with React and Vite
- Styled with Tailwind CSS
- Uses shadcn/ui components
- Fully static - no backend required
- Optimized for GitHub Pages deployment
